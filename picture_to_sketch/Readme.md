## Readme



本项目实现从彩色图片到素描图片的转换。



只需要重新修改输入和输出的地址就可以了。



**使用**：

1. 修改路径

```python
input_path = r'C:\Users\Mr.Reliable\Desktop\蓝版.jpg'  #输入彩色图片
output_path = "C:/Users/Mr.Reliable/Desktop/sketch.jpg"
```

2. 执行文件

```python
python sketch.py
```
